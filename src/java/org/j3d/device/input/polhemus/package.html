Implementations of InputDevices suitable for the Polhemus devices - principally the line of *Trak line of trackers.

This code is a mildly modified form of the implementation provided by Satoshi Konno and the CyberVRML work. His work is licensed as effectively public domain. This code has taken it, fixed a number of bugs in the IsoTrak II code documented and placed under the j3d.org package heirarchy.
